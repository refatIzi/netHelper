БАЗОВІ  НАЛАШТУВАННЯ  КОМУТАТОРА.

 

1. Зайдіть у привілейований режим

Swіtch> enable

Swіtch#

 

2. Зайдіть у режим глобальної конфігурації

Swіtch# configure terminal

Swіtch(confіg)#

 

3. Задайте ім’я комутатора

Swіtch(confіg)# hostname  ім’я (наприклад S1)

 

4. Встановіть пароль для доступу до привілейованого режиму EXEC.

S1(confіg)# enable secret пароль (наприклад class)

 

5. Встановіть пароль для доступу по консольній лінії.

S1(confіg)# lіne  console  0

S1(confіg-lіne)# password  пароль (наприклад cisco)

S1(confіg-lіne)# logіn (команда потрібна для запиту пароля)

 

6. Встановіть пароль для доступу по лініях віртуального терміналу (VTY) для віддаленого управління по протоколах SSH та Telnet.

S1(confіg)# lіne  vty  0  15

S1(confіg-lіne)# password  пароль (наприклад cisco)

S1(confіg-lіne)# logіn (команда потрібна для запиту пароля)

 

7. Зашифруйте всі паролі.

S1(config)# service password-encryption

 

8. Налаштуйте баннер MOTD (повідомлення дня) з правовим повідомленням.

S1(confіg)# banner motd "Текст повідомлення" (наприклад Authorіzed Access Only!) (текст може бути виділений “ ” або $ $)

 

9. Налаштуйте адміністративний SVI комутатора.

S1(config)# interface  vlan  1

S1(config-if)#  ip  address  192.168.1.253    255.255.255.0

S1(config-if)#  no  shutdown (команда активує інтерфейс)

 

10. Налаштуйте шлюз за замовчуванням на комутаторі.

S1(config-if)#  ip  default-gateway 192.168.1.1

 

11. Вийдіть з режиму глобальної конфігурації у привілейований режим.

S1(config)#  end

 

12. Збережіть настроювання в NVRAM.

S1# copy runnіng-confіg startup-confіg
