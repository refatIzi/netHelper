Маршрутизація між VLAN

Налаштування комутатора

Switch> enable 

Switch# configure terminal

Switch(config)# vlan 100 (створюємо VLAN №100)

Switch(config-vlan)# name PC0 (присвоюємо їй ім’я «PC0»)

Switch(config-vlan)# exit

Switch(config)# vlan 200 (створюємо VLAN №200)

Switch(config-vlan)# name PC1 (присвоюємо їй ім’я «PC1»)

Switch(config-vlan)# exit

Switch(config)# interface f0/1

Switch(config-if)# switchport mode access (перемикання порта в режим доступу)

Switch(config-if)# switchport access vlan 100 (приналежність порта до VLAN 100)

Switch(config-if)# exit

Switch(config)# interface f0/2

Switch(config-if)# switchport mode access (перемикання порта в режим доступу)

Switch(config-if)# switchport access vlan 200 (приналежність порта до VLAN 200)

Switch(config-if)# exit

Switch(config)# interface g0/1

Switch(config-if)# switchport mode trunk (перемикання порта в режим trunk)

Switch(config-if)# exit

Switch(config)# interface f0/10

Switch(config-if)#switchport trunk native vlan 5

 

Налаштування маршрутизатора

Router> enable 

Router# configure terminal

Router(config)# interface gigabitEthernet 0/0

Router(config-if)# no shutdown (включення интерфейсу)

Router(config-if)# exit

Router(config)# interface gigabitEthernet 0/0.100 (створення підінтерфейсу 100)

Router(config-subif)# encapsulation dot1Q 100 (вмикання інкапсуляції dot1Q та

                                                                               підключення до VLAN 100)

Router(config-subif)# ip address 192.168.0.1 255.255.255.0

Router(config-subif)# exit

Router(config)# interface gigabitEthernet 0/0.200 (створення підінтерфейсу 200)

Router(config-subif)# encapsulation dot1Q 200 (вмикання інкапсуляції dot1Q та

                                                                               підключення до VLAN 200)

Router(config-subif)# ip address 172.16.0.1 255.255.255.0

Router(config-subif)# exit
