НАЛАШТУВАННЯ  МАРШРУТИЗАЦІЇ.

 

1. Налаштування  статичної маршрутизації

Налаштування показані для Router0

Router0(config)# ip route 0.0.0.0 0.0.0.0 serial 0/0/0 (маршрут за замовчуванням

Router0(config)# ip route 20.20.20.0 255.255.255.0 serial 0/0/0 (статичний маршрут до мережі 20.20.20.0 через інтерфейс serial 0/0/0 – напряму підключений маршрут)

Router0(config)# ip route 172.16.0.0 255.255.255.0 serial 0/0/0

інший варіант

Router0(config)# ip route 20.20.20.0 255.255.255.0 10.10.10.2 (маршрут наступного переходу (рекурсивний маршрут))

 

2. Налаштування динамічної маршрутизації OSPF (пошук оптимального каналу)

Налаштування показані для Router0

R0(config)# router ospf 1 (вмикання протоколу OSPF, де 1 – номер процесу OSPF)

R0(config-router)# network 192.168.50.0    0.0.0.255  area 1 

R0(config-router)# network 10.10.00.0     0.0.0.255  area 1

R0(config-router)# network 30.30.30.0     0.0.0.255  area 1

 

Налаштування динамічної маршрутизації RIP

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

R0(config)# router rip

R0(config-router)# version 2

R0(config-router)# network 10.0.0.0

R0(config-router)# network 192.168.0.0

R0(config-router)# passive-interface fastEthernet 0/0 – інтерфейс, через який не відправляються дані маршрутизації
