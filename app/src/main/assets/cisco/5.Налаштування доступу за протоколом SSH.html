Налаштування доступу за протоколом SSH:

 

1. Створити користувача з ім'ям та паролем.

username netadmin secret Cisco

 

2. Налаштувати лінії віртуального терміналу.

line vty 0 15

password cisco12345

login local

transport input ssh

exec-timeout 60 (після 60 хвилин без руху з’єднання закривається)

exit

 

4. Створити доменне ім’я та ключ шифрування.

 

no ip domain-lookup (відключає пошук по DNS)

ip domain-name Cisco.net (ствоює доменне ім’я пристрою)

login block-for 120 attempts 2 within 30 (блокування на 120 секунд після 2-х невдалих спроб за 30 секунд)

crypto key generate rsa (генерує ключ шифрування)

1024 (довжина ключа в бітах)

 

Для доступу з ПК слід використати програму емуляції терміналу (наприклад PUTTY), або у командному рядку набрати ssh -l netadmin 192.168.0.10
