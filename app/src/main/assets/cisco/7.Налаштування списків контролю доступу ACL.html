Налаштування списків контролю доступу ACL

 

Налаштування стандартного нумерованого ACL

R1(config)# access-list 1 deny host 192.168.10.10

R1(config)# access-list 1 permit 192.168.10.0 0.0.0.255

R1(config)# interface serial 0/0/0

R1(config-if)# ip access-group 1 out

 

Налаштування стандартного іменованого ACL

R1(config)# ip access-list standart Filter_LAN1

R1(config-std-nacl)# deny host 192.168.10.10

R1(config-std-nacl)# permit any

R1(config-std-nacl)# exit

R1(config)# interface G0/0

R1(config-if)# ip access-group Filter_LAN1 out

 

Налаштування розширеного нумерованого ACL

R1(config)# access-list 110 deny tcp host 192.168.0.1 any eq 80

R1(config)# access-list 110 deny tcp 192.168.0.128 0.0.0.127 gt 1024 any eq 80

R1(config)# access-list 110 permit ip any any

R1(config)# interface fa0/0

R1(config-if)# ip access-group 110 in

 

Налаштування розширеного іменованого ACL

R1(config)# ip access-list extended MY_ACL

R1(config-ext-nacl)# deny tcp host 192.168.0.1 any eq 80

R1(config-ext-nacl)# permit ip any any

R1(config-ext-nacl)# exit

R1(config)# interface fa0/0

R1(config-if)# ip access-group MY_ACL in

 

Видалення ACL-списку

R1(config)# interface serial0/0/0

R1(config-if)# no ip access-group 1 out

R1(config)# no access-list 1

 

Перегляд ACL-списку

show running-config – переглянути всі ACL-списки

show access-lists – переглянути всі ACL-списки

show running-config | include access-list 110 – переглянути 110 ACL-список

show access-list 110 – переглянути ACL-список з номером 110

show ip intreface g0/0 – переглянути ACL-список на певному інтерфейсі

 

deny – забороняє доступ

eq – тільки пакети з цим номером порта

permit – дозволяє доступ

gt – тільки пакети з більшим номером порта

host – конкретний вузол

lt – тільки пакети з меншим номером порта

any – всі вузли

neq – тільки пакети не з даним номером порта

 

range – діапазон портів
